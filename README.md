
# 🎬 DesiCinema OS

DesiCinema OS is a browser-based movie explorer inspired by a desktop operating system. Instead of a traditional movie website, I wanted to experiment with a window-based interface where different features open as movable applications, making the experience feel more interactive.

The project focuses on Indian cinema, with a collection of popular Bollywood comedy films and a few fun utilities for discovering new movies.

## Features

* 🎲 **CineSpin**

  * Randomly recommends a movie from the available collection.

* 🔍 **Movie Database**

  * Browse movies by genre.
  * Filter movies instantly using the built-in selector.

* ⭐ **Celebrity Recommendations**

  * Select a celebrity and get a few movie recommendations associated with them.
  * Includes actors such as Salman Khan, Ranbir Kapoor, Ranveer Singh, Shah Rukh Khan, Hrithik Roshan, and Deepika Padukone.

* 🖥️ **Desktop-style Interface**

  * Draggable application windows.
  * Sidebar navigation.
  * Taskbar with a live system clock.
  * Modern dark-themed design inspired by desktop operating systems.

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript

No frameworks or external libraries were used. Everything runs directly in the browser.
